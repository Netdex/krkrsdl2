# 吉里吉里SDL2

吉里吉里SDL2は、macOSやLinuxなど、[SDL2](https://www.libsdl.org/)をサポートするプラットフォームで実行できる[吉里吉里Z](https://krkrz.github.io/)の移植版です。  

詳細については、次の Web ページを参照してください: https://krkrsdl2.github.io/krkrsdl2/
